# ระบบใบสำคัญจ่าย 🏗️📄

โปรเจกต์ระบบจัดการ **ใบสำคัญจ่าย (Payment Voucher)** สำหรับบริษัทรับเหมาก่อสร้าง พัฒนาด้วย **Microsoft Access** รองรับการบันทึกการจ่ายเงิน แยกตามแผนกและโครงการ พร้อมพิมพ์ใบสำคัญจ่ายได้ทันที

---

## 🔍 ภาพรวมของโปรเจกต์

ระบบถูกออกแบบมาเพื่อจัดการเอกสาร **Payment Voucher (PV)** ภายในบริษัทก่อสร้าง โดยรองรับการบันทึกรายการจ่ายเงิน ทั้งเงินสดและเช็คธนาคาร เชื่อมโยงกับแผนกและโครงการที่เกี่ยวข้อง พร้อมระบบสร้างเลขที่เอกสารอัตโนมัติ และพิมพ์รายงานใบสำคัญจ่ายฉบับสมบูรณ์

---

## 🛠️ เทคโนโลยีที่ใช้

- **Platform**: Microsoft Access 2013/2016 (.accdb)
- **Excel Integration**: `BahtText` function ผ่าน Microsoft Excel 15.0 Object Library (แปลงตัวเลขเป็นตัวอักษรภาษาไทย)
- **Font**: Angsana New / Cordia New / Calibri

---

## 📂 โครงสร้างฐานข้อมูล

### 🗃️ ตารางข้อมูลหลัก (Tables)

| ตาราง          | ฟิลด์หลัก                                                                                                         | คำอธิบาย                      |
| -------------- | ----------------------------------------------------------------------------------------------------------------- | ----------------------------- |
| **Department** | Department_ID, Department_NO, Department_Name                                                                     | ข้อมูลแผนกในบริษัท            |
| **Job**        | Job_ID, Job_No, Job_Name, Status                                                                                  | ข้อมูลโครงการ/งาน             |
| **PV**         | ID, VoucherNo, Date, Date_Bill, Paid_To, Cash, Bank, Detail, Cheque_No, Due_Date, Department_ID, Job_No, Paid_For | ใบสำคัญจ่ายหลัก               |
| **PV_Detail**  | PV_ID, Particulars, Bill_NO, Amount                                                                               | รายละเอียดรายการในใบสำคัญจ่าย |
| **Detail**     | ID, Particulars, BillNo, Amount                                                                                   | ตารางรายละเอียดย่อย           |

## 📄 รายงาน (Reports)

- **Report_PV_Bill** — ใบสำคัญจ่ายฉบับสมบูรณ์ ประกอบด้วย:
  - ชื่อและที่อยู่บริษัท
  - เลขที่ใบสำคัญจ่าย (VoucherNo) / วันที่
  - ชื่อผู้รับเงิน (Paid To) / วัตถุประสงค์ (Paid For)
  - ประเภทการจ่าย: เงินสด / เช็คธนาคาร + เลขที่เช็ค + วันครบกำหนด
  - ตารางรายการ: เลขที่บิล, รายการ, จำนวนเงิน
  - ยอดรวม (Total Amount) พร้อม **จำนวนเงินเป็นตัวอักษรภาษาไทย** (BahtText)

## ✨ ฟีเจอร์หลัก

- 📝 **บันทึกใบสำคัญจ่าย** — กรอกข้อมูลการจ่ายเงินทั้งเงินสดและเช็ค
- 🔢 **สร้างเลขที่เอกสารอัตโนมัติ** — ระบบ Auto-generate เลขที่ PV ตามปีและลำดับ
- 🏢 **แยกตามแผนก** — เชื่อมโยงการจ่ายกับแผนกที่รับผิดชอบ
- 🏗️ **แยกตามโครงการ** — ติดตามค่าใช้จ่ายแต่ละโครงการก่อสร้าง
- 💰 **รองรับหลายรายการ** — บันทึกได้หลายบิลต่อใบสำคัญจ่าย
- 🖨️ **พิมพ์ใบสำคัญจ่าย** — ออกรายงานพร้อมตัวอักษรภาษาไทย (บาทอักษร)
- 🔍 **ค้นหาข้อมูล** — ฟังก์ชัน Find Record ในฟอร์มหลัก
